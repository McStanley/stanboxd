# Stanboxd

The best Letterboxd clone out there.


## Tech Stack

**Front End:** React, React Router DOM, React Hot Toast

**Back End:** Firebase, Vercel Edge Functions, TMDB API


## Features

- Film reviews
- Film search
- User search
- User authentication
- Profile settings
- Password recovery
